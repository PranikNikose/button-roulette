# Button Roulette

Simple Spring Boot + React application.

## Backend

```bash
cd roulette-backend
mvn spring-boot:run
```

Runs on:

```text
http://localhost:8888
```

## Frontend

Create:

```bash
cp .env.example .env
```

Start:

```bash
cd roulette-frontend
npm install
npm start
```

Runs on:

```text
http://localhost:3000
```
